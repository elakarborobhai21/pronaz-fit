⚙️ One-time setup
Two things to get: a Gemini key (2 min) and a Google Sheets URL (~15 min). Do this once — works forever.
Part A — Gemini API key (free)
1
Go to aistudio.google.com → sign in with your Google account.
2
Click Get API key → Create API key. Copy it.
3
Paste in Profile → API setup → Gemini API key.
Part B — Google Sheets (free)
1
Go to sheets.google.com → new sheet → name it ProNaz Fit Data.
2
Click Extensions → Apps Script. Delete everything. Paste this code:
function doPost(e){ var d=JSON.parse(e.postData.contents); var ss=SpreadsheetApp.getActiveSpreadsheet(); var sh=ss.getSheetByName(d.sheet)||ss.insertSheet(d.sheet); if(sh.getLastRow()===0){sh.appendRow(d.headers);} sh.appendRow(d.row); return ContentService .createTextOutput(JSON.stringify({ok:true})) .setMimeType(ContentService.MimeType.JSON); } function doGet(e){ var d=e.parameter; var ss=SpreadsheetApp.getActiveSpreadsheet(); var sh=ss.getSheetByName(d.sheet); if(!sh)return ContentService .createTextOutput(JSON.stringify({rows:[]})) .setMimeType(ContentService.MimeType.JSON); var rows=sh.getDataRange().getValues(); return ContentService .createTextOutput(JSON.stringify({rows:rows})) .setMimeType(ContentService.MimeType.JSON); }
3
Click Deploy → New deployment → Web app → Execute as: Me → Who has access: Anyone → Deploy. Copy the URL and paste in Profile → Google Sheets URL.
Part C — iPhone home screen
1
Open the app URL in Safari (not Chrome).
2
Tap Share (box+arrow) → Add to Home Screen → Add. Done!
